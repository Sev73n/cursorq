- 采用 Monorepo 架构，根目录通过 `package.json` workspaces 管理 `packages/core`（TypeScript 核心逻辑）与 `apps/tauri`（Tauri 前端与后端）。
- `packages/core` 负责跨平台业务逻辑：通过 `sql.js` 读取 Cursor 本地 SQLite 登录态，调用非公开 API 获取用量，并计算日预算、节余银行及进度条配色。
- `apps/tauri` 分为 Rust 后端 (`src-tauri`) 与 TypeScript 前端 (`src`)。后端通过 `lib.rs` 暴露 Tauri Commands，处理系统托盘、DWM 窗口形状调整、远程内容同步及调用 Node.js 脚本执行刷新；前端负责 UI 渲染、吉祥物动画控制及调试模式交互。
- 依赖方向明确：Tauri 前端依赖 `@cursorq/core` 进行数据可视化计算，Rust 后端通过进程间通信与 Node.js 环境协作完成数据抓取。
